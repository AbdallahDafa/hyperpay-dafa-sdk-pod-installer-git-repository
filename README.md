# hyperpay-dafa-sdk-pod-installer-git-repository
 install hyper pay from dafa

# source pod global

Anyone who wants to use your pod should add this to their Podfile:

```
pod 'hyper_pay_installer', '1.0.10'
```

Or just:

````
pod 'hyper_pay_installer'
````
